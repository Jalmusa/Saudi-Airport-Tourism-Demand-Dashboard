Readme File – Dashboard
Introduction:
The Saudi Airport Traffic dashboard was developed using Power BI to explore and analyze air traffic activity across airports in Saudi Arabia.
The dashboard provides an interactive and visual experience that helps identify key insights related to passenger traffic, flight distribution, airport performance, and domestic versus international travel trends.
Installation:
• Ensure that Power BI Desktop is installed on your computer. (It can be downloaded for free from the official Power BI website.)
• Download the dashboard file (.pbix) from the provided link or email.
• Open the file using Power BI Desktop to explore and interact with the data.
Usage:
-Filters/Slicers:
Easily filter data by Airport, City, Month, and Flight Type to focus on specific traffic patterns and airport activities.
-Visualizations:
The dashboard includes multiple visual models such as:
• KPI Cards: Overview of total passenger traffic, peak traffic month, and airport with the highest traffic.
• Map Visual: Geographical visualization of airport traffic across Saudi Arabia.
• Donut and Pie Charts: Analysis of flight distribution by city and monthly
passenger share.
• Column Charts: Comparison of airport traffic performance across different
locations.
• Line Chart: Domestic versus international traffic trends over time.
• Scatter Chart: Airport traffic changes and performance overtime.
• Scatter Chart: Airport traffic changes and performance overtime.
• Decomposition Tree: Detailed breakdown analysis of passenger traffic data.
- Interactivity:
All visuals are dynamic and interconnected. Selecting elements in one chart will automatically filter the other visuals for deeper insights and easier analysis
Troubleshooting:
-Missing Visuals: Ensure all data fields are correctly loaded. If issues persist, refresh the data source inside Power BI Desktop.
-Performance Issues:
If the dashboard feels slow, ensure your device meets the recommended Power BI Desktop system requirements.
-Data Refresh:
This dashboard uses static data; no scheduled refresh is required unless the dataset is manually updated.
-Support:
For any issues or questions, please contact the project team.
Conclusion:
The Saudi Airport Traffic dashboard is designed to support analysts, researchers, and decision-makers in understanding aviation activity and passenger movement trends across Saudi Arabia.
Through interactive visuals and user-friendly filters, users can quickly identify important traffic patterns, compare airport performance, and gain valuable operational insights.
Feedback and suggestions for improving the dashboard are always welcome.
